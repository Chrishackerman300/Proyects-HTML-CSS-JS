Proyecto Carrito de Compras con JavaScript

En este proyecto creare un carrito de compras, donde el ususario podra agregar la n cantidad de cursos que quiera,
donde aparecera el nombre del curso, costo, cantidad de veces que lo tiene, y su imagen, si tiene cursos repetidos
tendra la n catidad de veces de esos curso, podra eliminar un curso manualmente y podra vaciar por completo el carrito 
cuando el quiera.

Todo esto utilizando puro JavaScript, HTML5 y CSS3.
